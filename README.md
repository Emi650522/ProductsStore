# ProductsStore
Esta API provee operaciones básicas para gestión de productos, autenticación con JWT y un diseño basado en separación de capas (controllers, services, models).
Además, utiliza Firebase Firestore como base de datos NoSQL.

Node.js + Express

Firebase Admin SDK

Firestore (Base de datos)

JSON Web Tokens (JWT) para autenticación

Estructura por capas:
Controllers
Services
Models
Routes

[Correr la Api]
git clone [(https://github.com/Emi650522/ProductsStore.git)]
cd ProductsStore

npm install
